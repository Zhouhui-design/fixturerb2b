##  登录流程

1. 访问 https://fixr2026.com/admin
2. 输入管理员密码
3. 点击"登录管理后台"
4. 验证通过后进入仪表板

##  修改密码

如果需要修改密码，请按以下步骤操作：

### 1. 生成新密码的哈希值

在终端执行：
```bash
node generate-admin-hash.js <your-new-password>
```

例如：
```bash
node generate-admin-hash.js "MyNewPassword123!"
```

### 2. 更新 .env 文件

将生成的哈希值复制到 `.env` 文件：
```env
VITE_ADMIN_PASSWORD_HASH=<your-new-hash>
```

### 3. 重新构建并部署

```bash
npm run build
bash deploy.sh
```

##  安全特性

### SHA-256 哈希加密
- 密码以哈希值形式存储在环境变量中
- 不会在代码中明文存储密码
- 即使有人查看源代码也无法获取密码

### Session 认证
- 登录状态保存在浏览器 sessionStorage
- 关闭浏览器后自动清除
- 不会被持久化存储

### 访问控制
- 未认证用户无法查看管理后台内容
- 只有输入正确密码才能访问
- 所有敏感信息已脱敏显示

##  安全建议

1. **定期更换密码**：建议每 3-6 个月更换一次
2. **使用强密码**：至少 12 位，包含大小写字母、数字和符号
3. **不要分享密码**：仅在必要时告知可信的 AI 助手
4. **监控访问记录**：定期检查 Supabase 数据库的访问日志
5. **保护 .env 文件**：确保 .env 已添加到 .gitignore

##  故障排除

### 问题：无法登录
- 检查密码是否正确（区分大小写）
- 确认 .env 文件中的哈希值是否正确
- 尝试清除浏览器缓存后重新登录

### 问题：登录后仍然看到登录页面
- 检查浏览器是否禁用了 JavaScript
- 确认 sessionStorage 是否正常工作
- 尝试使用无痕模式测试

### 问题：忘记密码
- 查看 .env 文件中的默认密码注释
- 或联系管理员重置密码
