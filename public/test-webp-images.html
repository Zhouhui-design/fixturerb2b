使用说明

在React项目中使用OptimizedImage组件:

import OptimizedImage from '@/components/ui/OptimizedImage'

// 基本用法
<OptimizedImage 
  src="/images/photo.jpg" 
  alt="Description" 
/>

// 首屏优先加载
<OptimizedImage 
  src="/images/hero.jpg" 
  alt="Hero"
  priority={true}
/>

// 指定宽高比
<OptimizedImage 
  src="/images/product.jpg" 
  alt="Product"
  aspectRatio="16/9"
/>
