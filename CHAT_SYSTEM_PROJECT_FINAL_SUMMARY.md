## 📊 系统架构

### 技术栈

**后端**:
- Node.js (Express)
- MongoDB (Mongoose)
- Socket.IO (实时通信)
- Multer (文件上传)
- Nodemailer (邮件发送)

**前端**:
- Vanilla JavaScript
- HTML5 / CSS3
- Web Audio API
- Responsive Design

**基础设施**:
- DigitalOcean VPS
- Nginx (反向代理)
- PM2 (进程管理)
- Let's Encrypt (SSL)

### 架构图

```
客户端 (Browser)
    ↓ HTTPS
Nginx (反向代理 + SSL)
    ↓
Node.js Server (PM2)
    ├─ Express API
    ├─ Socket.IO (WebSocket)
    ├─ Multer (文件处理)
    └─ Nodemailer (邮件)
    ↓
MongoDB (数据存储)
```

### 遇到的挑战

   - 解决方案: 懒加载 + 资源优化
