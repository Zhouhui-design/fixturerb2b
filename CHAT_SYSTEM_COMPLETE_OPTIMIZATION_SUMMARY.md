## 💡 故障排查

### 常见问题

**1. 夜间模式不生效**
```javascript
// 清除 localStorage
localStorage.removeItem('chat_theme');
// 刷新页面
location.reload();
```

**2. 消息撤回失败**
- 检查是否在2分钟内
- 确认是发送者本人
- 查看控制台错误信息

**3. 文件上传失败**
- 检查文件大小（≤10MB）
- 确认文件类型在白名单内
- 查看服务器日志：`pm2 logs chat-system`

**4. 邮件未发送**
- 检查 Resend API Key 配置
- 验证 ADMIN_EMAIL 设置
- 查看日志：`pm2 logs | grep email`
