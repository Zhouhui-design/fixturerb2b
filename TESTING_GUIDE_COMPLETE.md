## 🔍 常见问题排查

### 问题 1: 麦克风无法使用

**可能原因**:
- 浏览器权限被拒绝
- HTTPS 未启用

**解决方法**:
1. 检查浏览器地址栏是否有麦克风图标
2. 点击图标，允许麦克风权限
3. 确保使用 HTTPS（https://chat.fixr2026.com/）

---

### 问题 2: 语音/视频通话无法连接

**可能原因**:
- NAT 穿透失败（需要 TURN 服务器）
- 防火墙阻止

**解决方法**:
1. 尝试在同一网络下测试
2. 如果仍然失败，需要配置 TURN 服务器
3. 查看浏览器控制台错误信息

---

### 问题 3: 翻译不准确

**说明**: 
- 当前使用 MyMemory 免费 API
- 适合日常交流，专业术语可能不准确

**改进方案**:
- 如需更高准确度，可升级为 Google Translate API（付费）

---

### 问题 4: 语音翻译无法识别关键词

**解决方法**:
1. 确保发音清晰
2. 在安静环境下使用
3. 可以尝试说英文："start translation" / "finished"

## 🐛 已知限制

1. **WebRTC 通话**:
   - 目前仅使用 STUN 服务器
   - 跨 NAT 可能需要 TURN 服务器
   - 建议在相同网络下先测试

2. **翻译功能**:
   - MyMemory API 有每日配额限制
   - 高频率使用可能触发限流

3. **语音识别**:
   - 浏览器原生 STT 依赖网络连接
   - 某些浏览器可能不支持

4. **移动端兼容性**:
   - iOS Safari 对 WebRTC 支持较好
   - Android Chrome 完全支持
   - 部分旧版浏览器可能不支持某些功能
