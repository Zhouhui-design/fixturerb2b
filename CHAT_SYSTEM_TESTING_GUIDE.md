## 🐛 常见问题排查

### 问题1：客户端没有自动连接管理员

**解决方案**：
1. 清除浏览器缓存
2. 检查控制台错误（F12）
3. 确认 API 是否正常：
   ```bash
   curl https://chat.fixr2026.com/api/auth/admin?tenantId=fixturerb2b
   ```

### 问题2：管理员无法登录

**解决方案**：
1. 确认用户名和密码正确
2. 检查后端服务是否运行：
   ```bash
   pm2 status
   ```
3. 查看错误日志：
   ```bash
   pm2 logs chat-system
   ```

### 问题3：消息无法发送

**解决方案**：
1. 检查 WebSocket 连接（F12 → Network → WS）
2. 确认 Socket.IO 连接成功
3. 查看服务器日志
