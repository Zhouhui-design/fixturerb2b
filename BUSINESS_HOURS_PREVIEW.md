## 🎨 Design Details

### Card 1: Human Support
- **Background:** Gray (`bg-secondary`)
- **Border:** 4px solid on left (primary color)
- **Icon:** 👥 (people emoji, 2em size)
- **Title:** "Live Support Hours (Human Team)" - Bold, large
- **Content:** Schedule details - Regular text, muted color

### Card 2: AI Support
- **Background:** Blue gradient (`from-blue-50 to-indigo-50`)
- **Border:** 4px solid blue on left
- **Icon:** 🤖 (robot emoji, 2em size)
- **Title:** "Outside These Hours?" - Bold, large, blue color
- **Content:** AI availability message - Blue text, includes "24/7" in bold

## ⚠️ Common Issues & Solutions

### Issue: Still seeing old single card
**Solution:** Hard refresh browser (Ctrl+Shift+R) or clear cache completely

### Issue: Icons not showing
**Solution:** Ensure browser supports emoji (most modern browsers do)

### Issue: Blue gradient not visible
**Solution:** Check if browser supports CSS gradients (all modern browsers do)

### Issue: Text overlapping or cut off
**Solution:** This shouldn't happen with current responsive design. Report if seen.

### Issue: Wrong language displaying
**Solution:** Check language selector in top navigation, ensure it's set correctly
