## 🚀 开通步骤（5分钟搞定，全程免费）

### 1️⃣ 开通智能语音交互服务

**访问地址**: https://ai.aliyun.com/nls/asr

**操作步骤**:
1. 点击「立即开通」
2. 用支付宝/淘宝账号扫码登录
3. 免费开通
4. ✅ **新用户赠送3个月免费额度（每月50小时）**

---

### 2️⃣ 获取 AccessKey

**访问地址**: https://ram.console.aliyun.com/manage/ak

**操作步骤**:
1. 鼠标悬停右上角头像
2. 点击「AccessKey管理」
3. 选择「继续使用AccessKey」
4. 点击「创建AccessKey」
5. **保存以下两个参数**：
   - `AccessKey ID` (格式: `LTAI5t...`)
   - `AccessKey Secret` (格式: `xxxxxxxx...`)

⚠️ **重要**: AccessKey Secret 只显示一次，请立即保存！

---

### 3️⃣ 获取语音应用 APPKEY

**访问地址**: https://nls-portal.console.aliyun.com/app

**操作步骤**:
1. 点击「创建项目」
2. 输入项目名（例如：chat-asr）
3. 选择「语音识别」
4. 创建完成后就能看到 **AppKey**
5. **保存 AppKey** (格式: `xxxxxxxx...`)
