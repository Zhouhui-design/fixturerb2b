## ⚠️ 待完善功能

### 语音转文字（需要后端支持）

**当前状态**: 前端框架已完成，需要后端集成 STT 服务

**推荐方案**:
1. **OpenAI Whisper API** - 准确率高，支持多语言
2. **Google Cloud Speech-to-Text** - 实时性好
3. **Azure Speech Service** - 企业级服务

**实现步骤**:
```javascript
// 1. 前端上传音频文件到后端
const formData = new FormData();
formData.append('audio', audioBlob);

// 2. 后端调用 Whisper API
const response = await openai.audio.transcriptions.create({
  file: audioFile,
  model: "whisper-1",
  language: "zh"
});

// 3. 返回转录文本
return { text: response.text };
```

**预计工作量**: 2-4 小时
