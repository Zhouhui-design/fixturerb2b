# 🌐 智能语音消息翻译功能使用指南

## 📋 功能概述

这是一个**智能语音翻译系统**，可以自动将聊天中的语音消息进行：
1. **语音识别（STT）** - 将语音转换为文字
2. **语言翻译** - 将文字翻译成目标语言
3. **文字转语音（TTS）** - 播放翻译后的语音

---

## 🎯 核心场景

### 场景 1: 我听对方的语音（对方说英语，我看中文）

**配置**:
- 对方的语音语言：英语
- 我想看到的文字：中文

**操作流程**:
```
对方发送英语语音 
  ↓
我点击语音消息的"三个点"图标
  ↓
选择"智能翻译"
  ↓
系统自动：
  1. 识别英语语音 → 英文文字
  2. 翻译英文 → 中文文字
  3. 显示中文译文
```

**结果**: 我看到中文文字，理解对方的意思

---

### 场景 2: 我发送语音给对方（我说中文，对方看英文）

**配置**:
- 我的语音语言：中文
- 对方想看到的文字：英文

**操作流程**:
```
我发送中文语音
  ↓
我点击自己语音消息的"三个点"图标
  ↓
选择"智能翻译"
  ↓
系统自动：
  1. 识别中文语音 → 中文文字
  2. 翻译中文 → 英文文字
  3. 播放英文语音（TTS）
  4. 显示中英文对照
```

**结果**: 对方听到英文语音，看到英文文字

## ⚙️ 配置步骤

### 第 1 步：打开翻译设置

1. 访问 https://chat.fixr2026.com/
2. 登录并进入聊天
3. 点击输入框上方的 **"🌐"** 翻译按钮

### 第 2 步：选择"语音消息"标签

在翻译弹窗中，点击底部的 **"💬 语音消息"** 标签

### 第 3 步：配置语言偏好

#### 👤 本人设置
- **我的语音语言**: 您说话使用的语言（例如：中文）
- **我想看到的文字**: 您希望看到的翻译文字语言（例如：中文）

#### 👥 聊天对象设置
- **对方的语音语言**: 对方说话使用的语言（例如：英语）
- **对方想看到的文字**: 对方希望看到的翻译文字语言（例如：英文）

### 第 4 步：保存配置

点击 **"保存配置"** 按钮

✅ 配置已保存到本地浏览器，下次自动加载

## 🎮 使用方法

### 方法 1: 翻译对方的语音

1. 收到对方的语音消息
2. 将鼠标悬停在语音消息上（或点击消息）
3. 点击右上角的 **"⋯"** 三个点图标
4. 选择 **"🌐 智能翻译"**
5. 等待处理（约 2-5 秒）
6. 查看翻译结果弹窗

**翻译结果包含**:
- 原文（对方语言的原始文字）
- 译文（翻译成您的语言）

---

### 方法 2: 翻译自己的语音

1. 发送语音消息后
2. 在自己的语音消息上点击 **"⋯"** 三个点图标
3. 选择 **"🌐 智能翻译"**
4. 等待处理
5. 系统会：
   - 显示中英文对照
   - 自动播放翻译后的语音（TTS）

## 💡 使用技巧

### 技巧 1: 快速切换语言配置

如果您经常与不同语言的人聊天：
1. 为每个聊天对象保存不同的配置
2. 使用浏览器的多个配置文件
3. 或者手动修改配置后再翻译

### 技巧 2: 提高识别准确率

- 在安静环境下录音
- 说话清晰、语速适中
- 避免背景噪音
- 使用高质量麦克风

### 技巧 3: 缓存机制

- 相同的翻译会自动缓存
- 第二次翻译相同内容会立即显示
- 减少 API 调用次数

## ❓ 常见问题

### Q1: 为什么翻译失败？

**可能原因**:
1. 网络连接问题
2. 音频文件格式不支持
3. 语音太短或太模糊
4. API 配额用尽

**解决方法**:
- 检查网络连接
- 重新录制清晰的语音
- 稍后重试

---

### Q2: 翻译不准确怎么办？

**说明**:
- 当前使用 MyMemory 免费 API
- 适合日常交流，专业术语可能不准确

**改进方案**:
- 升级为 Google Translate API（付费）
- 或使用百度翻译 API

---

### Q3: TTS 播放没有声音？

**可能原因**:
1. 浏览器不支持 Web Speech API
2. 系统音量静音
3. 浏览器权限被拒绝

**解决方法**:
- 检查系统音量
- 允许浏览器播放音频
- 使用 Chrome/Edge 浏览器

---

### Q4: 配置保存后丢失了？

**说明**:
- 配置保存在浏览器的 localStorage
- 清除浏览器数据会丢失配置

**建议**:
- 不要清除浏览器缓存
- 或者记住配置内容，需要时重新设置

## 🔧 技术细节

### 使用的技术栈

1. **语音识别**: 阿里云 ASR（智能语音交互）
   - 前 3 个月免费
   - 每月 50 小时额度
   - 高精度识别

2. **文字翻译**: MyMemory Translation API
   - 免费版本
   - 每日 1000 次请求
   - 支持 9 种语言

3. **文字转语音**: Web Speech Synthesis API
   - 浏览器原生
   - 无需额外 API
   - 支持多种语言
