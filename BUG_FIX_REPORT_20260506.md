### 问题 14: 翻译功能 🔄

**需求分三个部分**:

#### 14.1 文字翻译
- 选择输入语言和输出语言
- 输入文字后同步翻译
- 点击发送翻译后的文字

**实现方案**:
- 使用 MyMemory 免费翻译 API
- 前端实时翻译预览
- 预计工作量: 3-4 小时

#### 14.2 语音通话翻译
- 说话前说"开始翻译"
- 系统录音 → 转文字 → 翻译 → 转语音
- 说完"发言完毕"后播放翻译结果

**实现方案**:
- 语音识别: 阿里云 ASR（已配置）
- 翻译: MyMemory API 或 Google Translate API
- 文字转语音: Web Speech API（浏览器原生）
- 预计工作量: 1-2 天

#### 14.3 视频通话翻译
- 同语音通话翻译
- 在视频通话中实时翻译

**实现方案**:
- 复用语音通话翻译逻辑
- 添加视频流处理
- 预计工作量: 0.5 天
